# Media Critic

A simple project built to users be able to post their book, movie, or game critics online.

## Built with

- Ruby 3.2.2
- Rails 7.0.8

## Installing

```
$ git clone git@github.com:ramos-ph/cm42-media-critic.git
$ cd cm42-media-critic
$ cd backend
$ bundle
```

## Running the server

```
$ cd backend
$ bin/rails server
```
